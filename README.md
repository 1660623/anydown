# anydown

Markdownによる軽量予定管理ツールです。

2ペインのシンプルなMarkdownエディタに、カンバンやガントなどのコンポーネントがついています。  
それぞれのコンポーネントは **双方向編集** に対応しています。  
保存先はローカルストレージです。
